Información del Curso

Descripción

Desarrollo Web cubre fundamentos y técnicas modernas para crear aplicaciones web interactivas y responsivas.

Contenidos

HTML5, CSS3, Flexbox, Grid Layout, diseño responsivo, JavaScript y manipulación del DOM.

Objetivos

Desarrollar competencias para diseñar y construir sitios web modernos y optimizados para diferentes dispositivos.

Guías de trabajo

Guía 01: Fundamentos de HTML y CSS

Introducción a HTML5 y CSS3. Exploración de diferentes formas de aplicar estilos y trabajo con estructura semántica. Incluye creación de una página personal sobre una serie.

  • Estilos embebidos
  • Estilos en etiqueta
  • Países de Centroamérica
  • Página de serie

Guía 02: Diseño y Maquetación

Técnicas de maquetación con CSS. Trabajo con modelo de caja, posicionamiento y diseño de layouts. Ejercicios incluyen página de película, tablero y biografía.

  • Película
  • Tablero
  • Biografía

Guía 03: CSS Avanzado y Responsive Design

Técnicas avanzadas incluyendo Flexbox, Grid Layout y Media Queries. Desarrollo de navegación horizontal, layouts adaptativos y análisis de sitios web.

  • Ejercicio 2
  • Ejercicio 3
