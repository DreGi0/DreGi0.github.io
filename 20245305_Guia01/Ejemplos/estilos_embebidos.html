Estilos embebidos en las etiquetas HTML

Usar estilos CSS embebidos en las etiquetas HTML es el peor y el menos utilizado de los m√©todos para agregar reglas de estilo. No es nada recomendable por la dificultad que implica realizar un cambio a posterior.

Esta forma de incluir CSS directamente en los elementos HTML solamente se utiliza en determinadas situaciones en las que se debe incluir un estilo muy especifico para un solo elemento concreto.

Para insertar un estilo CSS en un elemento se utiliza la propiedad style de HTML, que en ningun momento se debe confundi con la etiqueta <style>
